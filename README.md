## FEATURES
**MaxClear** is designed to keep your server running smoothly by reducing lag caused by dropped items and excessive entities. Everything is fully customizable!

- **Automatic & Manual Clearing**
  - Set up automatic entity clears at regular intervals.
  - Execute manual clears anytime using admin commands.
- **Advanced Anti-Farms (Chunk Limits)**
  - Automatically limit the maximum number of entities allowed per chunk to prevent extreme lag from mob farms.
- **Smart Entity Filtering**
  - **Target List:** Define exactly which entities should be removed (e.g., Drops, Arrows, TNT, Experience Orbs).
  - **Safe List (Exclusions):** Protect important entities like Villagers, Armor Stands, Minecarts, Boats, and Item Frames.
- **Item & Spawn Protection**
  - **Named Items:** Protect dropped items that have custom names (prevents players from losing valuable custom gear).
  - **Spawn Radius:** Set a safe-zone distance from the spawn point where entities will never be cleared.
- **Multi-World Support**
  - Choose specific worlds where the plugin should run, or apply the clear globally across all worlds.
- **100% Customizable Messages & Warnings**
  - Custom chat prefix and Minecraft color code (`&`) support.
  - Configurable warning announcements before a clear happens to give players time to pick up their items.
- **Built-in Update Checker**
  - Automatically notifies server administrators on join when a new update is available.
